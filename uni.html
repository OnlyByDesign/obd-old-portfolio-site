A particular challenge with this project was the constant back and forth and the tight deadline we were under - we had one month from inception to completion for the entire app, but delivered with time to spare.

Constant changes to code, videos, and embedded content meant a complete understanding of the platform was required, as well as a few others. Technologies that were used ranged from simple videos and animations to 3D assets built in Sketchfab and independantly.

Bringing in custom assets from Sketchfab, a scene was built and can by cycled through by custom buttons built using the Sketchfab API and vanilla JavaScript.

Using jQuery Sequencer, I produced a scrolling animation out of hundreds of PNG images, seamlessly moving together on touch. Each frame you see in an individual PNG image.
