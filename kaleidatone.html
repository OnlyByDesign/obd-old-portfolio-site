Web design

Front-end

Kaliedatone Media is a small collective

of sound designers, musicians, and the like providing audio services to large companies and enterprises. Coming to me for a simple website built through WordPress, I provided just that. Using on-mouse parallax, page transitions, and a clear-cut user interface - the site is simple, easy to navigate and understand, and delivers its message effectively; less is best.

Art Director

Gavin Davies

Each page features stunning photography with a slight parallax effect, pushing the narrative that minimalism can tell a story more than any amount of words on a page.

Using WordPress and Advanced Custom Fields, the site was built with the user in mind; easily adjustable, with content that can be changed out with a few simple clicks.

Fully responsive, the mobile experience was meant to be separate yet together; the interface easily navigable and clear.
